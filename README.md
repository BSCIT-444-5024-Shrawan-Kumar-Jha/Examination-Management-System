# 🎓 Online Examination Management System

A full-stack web application designed to conduct and manage online examinations efficiently. This system provides separate dashboards for students and teachers, enabling seamless exam creation, participation, and evaluation.

---

## 🚀 Features

### 👨‍🏫 Teacher / Admin Panel
- Create and manage exams
- Add and manage questions
- View student performance and analytics
- Generate PDF reports of exam statistics
- View and manage student feedback
- Send notifications to students

### 👨‍🎓 Student Panel
- Register and login securely
- Attempt exams online
- View scores and performance
- Submit feedback to teachers

### 🔐 Authentication & Security
- JWT-based login system
- OTP-based Forgot Password system
- Secure password hashing (bcrypt)

### 📄 PDF Report Generation
- Convert exam statistics into downloadable PDF
- Uses `jsPDF` and `html2canvas`
